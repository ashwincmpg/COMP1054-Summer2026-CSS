## 🚀 How to Use This Repository

### 1. Fork the Repository
- Click **Fork** (top-right of this page)  
- This creates your own copy of the repository  

---

### 2. Clone Your Fork
- Copy the URL of **your fork** (not the instructor repo)  
- In GitHub Desktop:  
  `File → Clone Repository → Paste URL`  

---

### 3. 🌿 Create a Branch (Required)
**Do NOT work on the `main` branch.**

Before starting any lab or assignment:
- Create a new branch  
- Use simple names like:
  - `lab-01`
  - `assignment-02`

---

### 4. Do Your Work
- Work only in your branch  
- Test your files in the browser  

---

### 5. Commit & Push
- Commit with clear messages (e.g., “Completed Lab 1”)  
- Push your branch to GitHub  

---

### 6. Submit Your Work
- Keep your work on your lab/assignment branch  
- Submit as instructed (do not merge into `main`)  

---

## 🔄 Keeping Your Fork Up to Date

When the instructor adds updates, follow this order:

### In GitHub Desktop

1. **Fetch/Pull origin**  
   - Sync your local repo with your GitHub fork  

2. **Fetch upstream**  
   - Get the latest instructor changes  

3. **Merge `upstream/main` into your current branch**  
   - Make sure you are on your lab/assignment branch  
   - Do **not** merge into `main`  

4. **Push origin**  
   - Update your fork on GitHub
